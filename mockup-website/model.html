🔬 Model Development Process

1. Data Preparation

Clean dataset of 381,109 records split into training and testing sets with stratified sampling to maintain class distribution balance (87.7% vs 12.3%).

2. Class Balancing with SMOTE

Applied SMOTE (Synthetic Minority Oversampling Technique) to handle severe class imbalance during training, creating synthetic examples of interested customers.

3. Feature Engineering

Comprehensive feature transformation including age categorization, premium quantiles, vehicle age mapping, and risk profile creation.

4. Model Training & Validation

Trained multiple algorithms with cross-validation, hyperparameter tuning, and rigorous performance evaluation across business-critical metrics.

5. Performance Optimization

Fine-tuned XGBoost with scale_pos_weight adjustment and class balancing to optimize for business objectives and maximize customer capture.

6. Model Deployment

Implemented production-ready prediction pipeline with real-time scoring capabilities and continuous monitoring for sustained performance.
